ng-repeat Directive

The ng-repeat directive repeats a set of HTML, a given number of times.
If we have an collection of objects, the ng-repeat directive is perfect for making a HTML table, displaying one table row for each object, and one table data for each object property.

Abowe we have a ng-repeat directive example.
The data in the table comes from the employees array which is located in the controller (myController in script.js file).
We access to the employees array via the $scope.
Then, to use ng-repeat directive we need to specify refference name for each element in the array. In our case let's assign the name "employee".
