Data Sorting

The orderBy filter allows us to sort an array.
By default, strings are sorted alphabetically, and numbers are sorted numerically.

The above method allows us to organize data according to the selected option.
For this purpose we will use the select element.
$scope.sortColumn = 'name' in the controller means that we have assigned a default sort by the name.
Then in the select tag, we specify the options with which we want to sort the tables using + for ascending and - for sescending.
Now we just set the sort method as sortColumn for ng-repeat in our table.
