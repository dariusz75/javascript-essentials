Data Sorting by Search filter

The orderBy filter allows us to sort an array.
By default, strings are sorted alphabetically, and numbers are sorted numerically.

The way the search filter works is very simple.
In the input tag, we specify ng-model under any name. In our case it is a "searchText".
Next we place the "searchText" directive as a filter in our ng-repeat directive
ng-repeat="employee in employees | filter:searchText"
In this way we search all the cells of the table. If we are interested in searching within only one column(eg. name column), we can specify it in the model.
ng-model="searchText.name"
