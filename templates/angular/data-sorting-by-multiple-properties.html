Data Sorting by Multiple Properties

The orderBy filter allows us to sort the data by multiple properties

To search with exact match we need to add checkbox with ng-model specified. In our case: ng-model="exactMatch"
Next we need to pass this ng-model to the filter in ng-repeat directive.
ng-repeat="employee in employees | filter:searchText:exactMatch"
