Data Binding

Data binding in AngularJS is the synchronization between the model and the view.
AngularJS applications usually have a data model. The data model is a collection of data available for the application.

Two-way Data Binding

When data in the model changes, the view reflects the change, and when data in the view changes, the model is updated as well.
This happens immediately and automatically, which makes sure that the model and the view is updated at all times.

ng-model = "message" comes from js file and it contains text - Hallo Angular!
Therefore, in the first input field and in the first paragraph, the same text appears.
By entering the text in the input field, we change its value in the paragraph, because we make two-way-databinding and Angular updates the model.
Then we have another input field and model ng-model = "nonFromController". This model does not come from the controller so it has no assigned text.
Angular will create and update the model and text that we enter in the second input box appears in the second paragraph.
