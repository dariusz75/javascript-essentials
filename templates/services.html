Services

In AngularJS, a service is a function, or object, that is available for, and limited to, AngularJS application.
In AngularJS we can make our own service, or use one of the many built-in services.

Let's go through the example

We declared ng-models: ng-model="input" and ng-model="outut"
We also specified what we want to do when user clicks the button.
For this we used ng-click directive with assigned function - ng-click="transformString()
Now we need to provide the data to the function.
In our case it is an input data entered by the user - ng-click="transformString(input)
Finally we need to define the transformString() function in the controller.
